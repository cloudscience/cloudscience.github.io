VPC Watch

VPC Watch monitors your AWS Virtual Private Cloud environment and notifies you when changes occur. VPC Watch provides:

  • Signed profiles of your VPCs
  • Change monitoring of over 50 VPC items
  • Reliable audit logging for forensic research

Learn more about VPC Watch.

API Sentry

API Sentry provides real-time security and financial controls to AWS APIs for enterprise users and AWS resellers. API Sentry can:

  • Enforce financial limits on AWS accounts
  • Apply sophisticated security to AWS access
  • Create fine-grained AWS account audits

Learn more about API Sentry.
