VPC Watch

Know when things change in your AWS Virtual Private Cloud.

The Problem

The AWS Virtual Private Cloud (VPC) is a powerful tool in your cloud portfolio. You need a way to know when changes to your VPC environment occur so you can prevent programming mistakes and malicious users from costing you money.

Your Solution

VPC Watch monitors your VPC 24/7/365, alerting you when changes occur so you can respond to them.

How VPC Watch Works

VPC Watch monitors over 25 specific data points for change, including:

  • Security groups
  • Firewall rules
  • VPN connections
  • Routing tables
