API Sentry

Real-time control for AWS API users.

The Problem

The AWS API provides your organization and its clients with a powerful way to programmatically manage your cloud resources. But as a reseller or enterprise organization, you want to ensure that faulty programs or poor communication don't result in overspending, potentially making you liable for those resource costs.

Your Solution

API Sentry applies authorization and financial checks to each of your AWS API calls, ensuring that only the changes you authorize are made, and only the dollars you approve are spent.

How It Works

API Sentry is a smart proxy that can check each AWS API call and enforce:

  • Spending limits before changes are made
  • Granular security permissions
  • Detailed audit logging for every change request

API Sentry is transparent -- you won't have to change your programs or vendor tools to use it. Just configure a new AWS API endpoint and you're protected.
