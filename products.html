Our Products

Let CloudScience provide you the tools to protect and manage your cloud environment.

VPC Watch

VPC Watch is an automated monitoring system that lets you know when something has changed in your AWS Virtual Private Cloud. VPC Watch can identify:

  • New server instances
  • Network and firewall changes
  • Security policy modifications

Learn more about VPC Watch.

API Sentry

API Sentry applies rules in real-time to AWS API calls. API Sentry allows resellers and enterprises to:

  • Apply financial limits to what your clients and users can spend
  • Apply fine-grained authorization and access controls to AWS resources
  • Create detailed audits of all AWS changes

Learn more about API Sentry.
